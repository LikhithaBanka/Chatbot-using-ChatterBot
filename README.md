# 🤖 MyBot - AI Chatbot using Flask and ChatterBot

## 📌 Project Description

MyBot is a web-based AI chatbot developed using Python, Flask, and ChatterBot. It simulates human-like conversations through an interactive chat interface and can respond to greetings, common questions, and jokes.

## 🛠 Technologies Used

**Backend**

* Python
* Flask
* ChatterBot
* SQLite
* SpaCy
* PyYAML

**Frontend**

* HTML5
* CSS3
* JavaScript

## ⚙️ Features

* Interactive chat interface
* Dark and Light mode
* Real-time messaging
* Timestamps
* Clear chat option
* Custom responses
* Automatic scrolling

## 🔄 How It Works

1. Users enter messages in the chat interface.
2. Flask sends the input to ChatterBot.
3. ChatterBot processes the message and generates a response.
4. The response is displayed instantly on the webpage.

## 📚 Applications

* Customer Support Systems
* FAQ Bots
* Educational Assistants
* Personal Virtual Assistants
